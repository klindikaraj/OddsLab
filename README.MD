## рҹ“– Descrizione

OddsLab ГЁ una piattaforma web che **identifica errori nelle quote dei bookmaker**
confrontandole con modelli statistici proprietari. Il sistema:

1. **Raccoglie** quote in tempo reale da 20+ bookmaker tramite The Odds API
2. **Calcola** la probabilitГ  reale con modelli Poisson (calcio) ed Elo (basket)
3. **Trova** le Value Bets вҖ” scommesse dove il bookmaker paga piГ№ del dovuto
4. **Suggerisce** quanto puntare con il **Criterio di Kelly**
5. **Genera** report pre-match narrativi tramite **Intelligenza Artificiale** (OpenAI)
6. **Aggiorna** i risultati tramite football-data.org (calcio) e balldontlie.io (NBA)

## рҹ“Ҡ FunzionalitГ  Principali

### рҹ”Қ Value Bet Finder
Confronta la probabilitГ  calcolata dal modello con la probabilitГ  implicita
delle quote. Se `(prob_modello Г— quota) - 1 > 2%`, segnala una Value Bet.

### рҹ§® Kelly Criterion
Calcola automaticamente quanto puntare in base al vantaggio trovato:

```
f* = (p Г— b - 1) / (b - 1)
```

Con safety cap al 10% e supporto per Full / Half / Quarter Kelly.

### рҹ“Ҳ Modello Poisson (Calcio)
Usa la distribuzione di Poisson per calcolare la probabilitГ  di ogni
risultato possibile, basandosi sulle medie gol fatti/subiti delle squadre.

### рҹҸҖ Modello Elo (Basket)
Sistema di rating che predice la probabilitГ  di vittoria basandosi
sulla differenza di punteggio Elo tra le due squadre.

### рҹӨ– Report IA
OpenAI GPT-4o-mini analizza i dati statistici e genera un report
pre-match di ~250 parole che spiega perchГ© una scommessa ГЁ una Value Bet.

### вҡҷпёҸ Automazione (cron)
La cartella `cron/` contiene script bash pronti per crontab:
- `fetch_odds.sh` вҖ” raccolta quote ogni 6 ore
- `run_models.sh` вҖ” previsioni + ricerca value bets
- `update_results.sh` вҖ” aggiornamento risultati e bankroll
